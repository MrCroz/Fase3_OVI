ETIQUETA NAV PARA DISEÑO WEB

La etiqueta nav permite crear una barra de navegación que almacena enlaces para moverse dentro de la página web o también puede contener enlaces a documentos u otras páginas web.

La etiqueta < nav>< /nav> es una buena forma de mejorar la accesibilidad de nuestra web y volverla más versátil a los buscadores de información dado que presentan las partes más relevantes de la web.
