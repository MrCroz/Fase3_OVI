ESTRUCTURA BODY PARA DISEÑO WEB

El cuerpo (body) del documento html es normalmente lo más importante. Es aquí donde debemos colocar el contenido de nuestra página que va a visualizar el usuario: texto, fotos, videos. El cuerpo está delimitado por las etiquetas < body >< /body >. y contiene sub etiquetas para seccionar el contenido que se va a mostrar al publico

Esta estructura interna dentro del body esta compuesta por las etiquetas < header>< /header>, < nav>< /nav>, < section>< /section>, < aside>< /aside> y < footer>< /footer>
