ETIQUETA HEADER PARA DISEÑO WEB

La etiqueta header contiene al encabezado de una sección o documento, habitualmente se colocan las portadas, los vínculos de navegación, los formularios de búsqueda, los logos, las tablas de contenidos, las introducciones, etc.
En muchos casos, los contenidos del encabezado de un documento son consistentes a lo largo de todo el sitio.

No se debe confundir el < header>< /header> que contiene elementos de introducción de contenido con el < head>< /head> que contiene metadatos usados por la web.
